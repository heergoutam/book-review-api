# 📚 Book Review API

A backend API to create and review books using **FastAPI**, **PostgreSQL**, **SQLAlchemy**, and **Alembic**.

## 🚀 Features

- ✅ Add and fetch books
- ✅ Add and view reviews for books
- ✅ Auto-generated Swagger docs at `/docs`

## 🛠️ Tech Stack

- FastAPI
- SQLAlchemy + Alembic
- PostgreSQL
- Pydantic v2
- Uvicorn (ASGI server)
